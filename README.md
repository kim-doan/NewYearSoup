## 🚀 사용 기술

**Back-End**
- Hasura GraphQL
- PostgreSQL
- Firebase Authentication
- Firebase Realtime Database
- Firebase Functions (node.js v10)
- Firebase Storage

**Front-End**
- Gatsby.js
- TypeScript(40%)
- redux
- redux-saga
- redux-toolkit
- axios
